# Supervision

## What?

- 受託制作会社の進行・管理を楽にするツール
- NOT a CMS

## Why?

納品物の品質が会社の価値となる受託制作。  
どんなメンバーで動いていようと、一定の品質を保てるようにするのがディレクターの仕事。  
でも、ディレクターの能力もピンキリ。  

そんな属人的Web制作に機械化を取り入れて一定の品質を手に入れよう。  
そして、ディレクターは＋αの価値を生み出そう。
